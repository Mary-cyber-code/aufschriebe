Mathematik: Differentialrechnung

1. Der Ableitungsbegriff

Die Ableitung einer Funktion an einer Stelle beschreibt die lokale Änderungsrate bzw. die Steigung der Tangente an den Graphen der Funktion in diesem Punkt.

Definition via Differentialquotient:

Man nähert sich der Tangentensteigung durch eine Sekantensteigung an. Die Formel lautet:

f'(x) = lim (h -> 0) [f(x + h) - f(x)] / h

2. Wichtige Ableitungsregeln

Potenzregel

Aus f(x) = xⁿ wird:

f'(x) = n · xⁿ⁻¹

Faktorregel

Ein konstanter Faktor bleibt beim Ableiten erhalten:

g(x) = c · f(x) → g'(x) = c · f'(x)

Mathematik: Differentialrechnung

3. Kurvendiskussion

Mithilfe der Ableitungen können wir charakteristische Punkte eines Graphen bestimmen:

  • Notwendige Bedingung für Extrempunkte: f'(x) = 0
  • Hinreichende Bedingung für ein Maximum: f'(x) = 0 und f''(x) < 0
  • Hinreichende Bedingung für ein Minimum: f'(x) = 0 und f''(x) > 0

Chemie: Säure-Base-Gleichgewichte

1. Definition nach Brønsted

Säure-Base-Reaktionen sind im Kern Protonenübertragungsreaktionen (Protolyse).

Säure (Protonendonator)

Gibt Protonen ($H^+$-Ionen) ab. Beispiel: Salzsäure ($HCl$).

Base (Protonenakzeptor)

Nimmt Protonen ($H^+$-Ionen) auf. Beispiel: Ammoniak ($NH₃$).

2. Der pH-Wert

Der pH-Wert ist der negative Dekadische Logarithmus der Oxoniumionen-Konzentration.

pH = -log₁₀[H₃O⁺]
